## 说明

### 目的
1. 练习three.js的使用
2. 新手的麻将练习程序，辅助理解麻将逻辑，如何快速做牌。
### 定位
现在将其定位于一个实现基本的逻辑。

如果能够坚持一直完善的话，希望能将其开发成为一个简单的人机麻将游戏（机器AI弱智的那种）。
